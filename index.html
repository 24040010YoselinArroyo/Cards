--- Conceptos básicos de JavaScript ---

Explora los fundamentos de JavaScript: Selectores, Eventos y Funciones.

Selectores

Los selectores en JavaScript permiten acceder y manipular elementos del DOM (la estructura HTML). Se usan para cambiar contenido, estilos o responder a interacciones.

const titulo = document.querySelector('h1');
titulo.style.color = 'blue';

🔹 Uso: document.querySelector(), getElementById()

Eventos

Los eventos permiten ejecutar acciones cuando el usuario interactúa con la página, como hacer clic, mover el mouse o escribir.

const btn = document.getElementById('boton');
btn.addEventListener('click', () => {
  alert('¡Evento activado!');
});

🔹 Uso: addEventListener()

Funciones

Las funciones agrupan bloques de código que se pueden reutilizar para realizar tareas específicas. Son esenciales en cualquier programa.

function saludar(nombre) {
  console.log('Hola ' + nombre);
}
saludar('UTC');

🔹 Uso: function, return, parámetros
